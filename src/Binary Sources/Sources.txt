This Sources file outlines where to find the third party tools (tools authored entirely by others) that are either used in YUMI or in the case of NSIS, used to create it.
The binarysources.zip file can be found at http://www.pendrivelinux.com/downloads/Sources/USB-Installers/YUMI/ and contains the sources for grub.exe, syslinux.exe, 7zip, and fat32format.exe. 

NSIS is the platform I used to create this tool!
